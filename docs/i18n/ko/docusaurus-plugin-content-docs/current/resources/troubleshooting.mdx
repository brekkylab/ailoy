# 문제 해결

## Python 또는 Node.js에서 Ailoy를 임포트할 수 없습니다.

### Windows 및 Linux

Ailoy는 **Vulkan 지원이 있는** GPU 드라이버가 올바르게 설치되어 있어야 합니다. Vulkan을 사용할 수 없거나 GPU 드라이버가 없거나 잘못 설치된 경우 동적 링크 중 누락된 심볼로 인한 임포트 시간 오류가 발생할 수 있습니다.

이를 해결하려면:

- GPU 드라이버가 올바르게 설치되어 있는지 확인하세요.
- `vulkaninfo`를 실행하여 Vulkan을 사용할 수 있는지 확인하세요 (아래 참조).

### macOS

Ailoy는 현재 **Apple Silicon (M1/M2/M3/M4...) 만** 지원합니다. Intel 기반 Mac은 GPU 및 API 기능이 없어 **지원되지 않습니다**.

자세한 내용은 [지원 환경](./supported-environments) 페이지를 참조하세요.

## Linux에서 코드가 매우 느리게 실행됩니다.

Linux에서 Ailoy는 Mesa에서 제공하는 CPU 기반 Vulkan 구현인 **llvmpipe**를 사용하도록 폴백될 수 있습니다. llvmpipe는 GPU 가속을 사용하지 않기 때문에 모델 실행이 매우 느립니다.

`vulkaninfo`를 실행하여 Vulkan 장치를 검사하세요:

```bash
$ vulkaninfo --summary

# `vulkaninfo`가 없는 경우:
$ sudo apt-get install vulkan-tools
```

다음과 같은 장치가 표시되면:

```bash
GPU1:
	apiVersion         = 1.4.305
	driverVersion      = 0.0.1
	vendorID           = 0x10005
	deviceID           = 0x0000
	deviceType         = PHYSICAL_DEVICE_TYPE_CPU
	deviceName         = llvmpipe (LLVM 20.1.2, 256 bits)
	driverID           = DRIVER_ID_MESA_LLVMPIPE
	driverName         = llvmpipe
	driverInfo         = Mesa 25.0.7-0ubuntu0.24.04.2 (LLVM 20.1.2)
	conformanceVersion = 1.3.1.1
	deviceUUID         = 6d657361-3235-2e30-2e37-2d3075627500
	driverUUID         = 6c6c766d-7069-7065-5555-494400000000
```

시스템이 llvmpipe Vulkan 장치를 사용하고 있는 것입니다.

llvmpipe를 제공하는 Mesa Vulkan 소프트웨어 드라이버를 제거하세요:

```bash
$ sudo apt-get remove mesa-vulkan-drivers
```

제거 후 `vulkaninfo`를 다시 실행하여 실제 GPU 드라이버가 사용되고 있는지 확인하세요.
