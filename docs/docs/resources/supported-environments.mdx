# Supported Environments

## For Agents with LLM APIs

You can create _your own agent_ with **Ailoy** using external APIs such as
OpenAI, Claude, Gemini or Grok.  
Your system doesn't need to meet the hardware requirements with these APIs.

## For Agents with AI running locally

**Local AI** is one of biggest feature of Ailoy that make it different from
other AI tools. To make it feasible, you might need to check the conditions for
the devices & environments to run AIs.

{/* prettier-ignore-start */}

- <SmallIcon src="https://www.microsoft.com/favicon.ico?v2" />
  Windows (x86_64, with
  <SmallIcon src="https://www.vulkan.org/user/themes/vulkan/images/favicon.png" />
  Vulkan)
  - **CPU**: Intel Skylake or newer (and compatible AMD), x86_64 is required
  - **GPU**: At least 8GB of VRAM and support for Vulkan 1.3
  - **NVIDIA/AMD/Intel graphic driver** that supports Vulkan 1.3 or higher
  - **OS**: Windows 11 or Windows Server 2022 (earlier versions may work but are not officially tested)
- <SmallIcon src="https://www.apple.com/favicon.ico" />
  macOS (Apple Silicon, with
  <SmallIcon src="https://developer.apple.com/assets/elements/icons/metal/metal-256x256.png" />
  Metal)
  - **Device**: Apple Silicon with Metal support
  - **Memory**: At least 8GB of unified memory
  - **OS**: macOS 15 or newer
- <SmallIcon src="https://upload.wikimedia.org/wikipedia/commons/3/35/Tux.svg" />
  Linux (x86_64, with
  <SmallIcon src="https://www.vulkan.org/user/themes/vulkan/images/favicon.png" />
  Vulkan)
  - **CPU**: Intel Skylake or newer (and compatible AMD), x86_64 is required
  - **GPU**: At least 8GB of VRAM and support for Vulkan 1.3
  - **NVIDIA/AMD/Intel graphic driver** that supports Vulkan 1.3 or higher
  - **OS**: Debian 10 / Ubuntu 21.04 or newer (this means, os with glibc 2.28 or higher)
- üåê Modern Web Browsers (wasm32, with
  <SmallIcon src="https://www.w3.org/2023/02/webgpu-logos/webgpu-notext.svg" />
  WebGPU)
  - **Browser**: Modern browsers with WebGPU support: 
    <SmallIcon src="https://www.google.co.kr/chrome/static/images/favicons/favicon-32x32.png" />
    **Chrome browsers** are recommended.
  - **GPU**: WebGPU with `"shader-f16"` support
  - It may not work on mobile devices with Qualcomm APs.

{/* prettier-ignore-end */}

AI models typically consume a significant amount of memory.

The exact usage depends on the model size, but we recommend at least **8GB of
GPU memory**. Running the Qwen 8B model requires at least **12GB of GPU
memory**. On macOS, this refers to unified memory, as Apple Silicon uses a
shared memory architecture.

Please refer to **[Available Models](available-models#vram-requirements)** for
more information.
