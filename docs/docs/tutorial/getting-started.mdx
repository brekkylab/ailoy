## Running a Simplest Agent

The **Language Model (LM)** is the core of every agent.  
An agent system operates based on a language model and its instructions, along
with various extensions that enhance its capabilities.

### Choosing Between API and Local Models

Ailoy lets you run language model in two different modes:

- **API Models** ‚Äì run models hosted by AI vendors (like OpenAI GPT or Anthropic
  Claude)
- **Local Models** ‚Äì run open-source models directly on your own machine

Each approach has its own strengths and trade-offs:

- **API Models**: These connect to external AI services over the internet. If
  you‚Äôre fine with token costs and prefer cloud-hosted models, this is the
  simplest option.
- **Local Models**: These run entirely on your own device, using open-source
  models. Choose this if you want full control, offline availability, or
  privacy.

Ailoy supports both seamlessly ‚Äî you can switch between them anytime with
minimal changes.

## Example

Let‚Äôs start with a simple example ‚Äî Ailoy‚Äôs version of ‚ÄúHello, World!‚Äù for LLMs.

In just a few lines of code, you‚Äôll run a model, send it a prompt, and print the
response.

### Using API models

{/* prettier-ignore-start */}

<CodeTabs>

```python
import ailoy as ai
import asyncio

async def main():
    model = ai.LangModel.new_stream_api(spec="OpenAI", model_name="gpt-4o", api_key="YOUR_API_TOKEN_KEY")
    agent = ai.Agent(model)
    query = ai.Message(role="user", contents="Please give me a short poem about AI")
    async for resp in agent.run([query]):
        if resp.aggregated:
            print(resp.aggregated.contents[0].text)

if __name__ == "__main__":
    asyncio.run(main())
```

```typescript
import { Agent, LangModel } from "ailoy-node";
import type { Message } from "ailoy-node";

async function main() {
  const lm = await LangModel.newStreamAPI(/* spec: */ "OpenAI", /* modelName: */ "gpt-5", /* apiKey: */ "YOUR_API_KEY");
  const agent = new Agent(lm);
  const query: Message = {
    role: "user",
    contents: [{ type: "text", text: "Please give me a short poem about AI" }],
  };
  for await (const resp of agent.run([query])) {
    if (resp.aggregated) {
      console.log(resp.aggregated.contents[0].text);
    }
  }
}

main().catch((err) => {
  console.error("Error:", err);
});
```

### Using local models

Here, we use the open-source model `Qwen3-0.6B`, running entirely on your own
device ‚Äî no API keys or internet connection required.

{/* prettier-ignore-start */}

<CodeTabs>

```python
import ailoy as ai
import asyncio

async def main():
    lm = await ai.LangModel.CreateLocal(model_name="Qwen/Qwen3-0.6B", progress_callback=print)
    agent = ai.Agent(lm)
    query = ai.Message(role="user", contents="Please give me a short poem about AI")
    async for resp in agent.run([query]):
        if resp.aggregated:
            print(resp.aggregated.contents[0].text)

if __name__ == "__main__":
    asyncio.run(main())
```

```typescript
import { Agent, LangModel } from "ailoy-node";
import type { Message } from "ailoy-node";

async function main() {
  const lm = await LangModel.newLocal(
    "Qwen/Qwen3-0.6B", console.log
  );
  const agent = new Agent(lm);
  const query: Message = {
    role: "user",
    contents: "Please give me a short poem about AI",
  };
  for await (const resp of agent.run([query])) {
    if (resp.aggregated) {
      console.log(resp.aggregated.contents[0].text);
    }
  }
}

main().catch((err) => {
  console.error("Error:", err);
});
```

:::info 
Don't be surprised if the output changes each time you run it.

An LLM's output includes a certain level of randomness, controlled by the `temperature` and `top_p`.
In ailoy, it can be adjusted with `InferenceConfig`.

If both `temperature` and `top_p` are set to 0, no randomness will occur.
