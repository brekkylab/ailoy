## Running a Simplest Agent

The **Language Model (LM)** is the core of every agent.
An agent system operates based on a language model and its instructions, along with various extensions that enhance its capabilities.

### Choosing Between API and Local Models

Ailoy lets you run language model in two different modes:

- **API Models** ‚Äì run models hosted by AI vendors (like OpenAI GPT or Anthropic Claude)
- **Local Models** ‚Äì run open-source models directly on your own machine

Each approach has its own strengths and trade-offs:

- **API Models**: These connect to external AI services over the internet. If you‚Äôre fine with token costs and prefer cloud-hosted models, this is the simplest option.
- **Local Models**: These run entirely on your own device, using open-source models. Choose this if you want full control, offline availability, or privacy.

Ailoy supports both seamlessly ‚Äî you can switch between them anytime with minimal changes.

## Example

Let‚Äôs start with a simple example ‚Äî Ailoy‚Äôs version of ‚ÄúHello, World!‚Äù for LLMs.

In just a few lines of code, you‚Äôll run a model, send it a prompt, and print the response.

### Using API models

<CodeTabs>

```python
import ailoy as ai
import asyncio

async def main():
    lm = ai.LangModel.CreateStreamAPI(spec=ai.APISpecification.OpenAI, model_name="gpt-4o", api_key="YOUR_API_TOKEN_KEY")
    agent = ai.Agent(lm)
    query = ai.Message(role=ai.Role.User, contents=[ai.Part.Text("Please give me a short poem about AI")])
    async for resp in agent.run([query]):
        if resp.aggregated:
            print(resp.aggregated.contents[0].text)

if __name__ == "__main__":
    asyncio.run(main())
```

```typescript
import { Agent, LangModel } from "ailoy";
import type { Message } from "ailoy";

async function main() {
    const lm = await LangModel.newStreamAPI(spec="OpenAI", modelName="gpt-4o", apiKey="YOUR_API_TOKEN_KEY");
    const agent = new Agent(lm);
    const query: Message = {role: "user", contents: [{ type: "text", text: "Please give me a short poem about AI" }]};
    for await (const resp of agent.run([query])) {
        if (resp.aggregated) {
            console.log(resp.aggregated.contents[0].text);
        }
    }
}

main().catch((err) => {
  console.error("Error:", err);
});
```

```rust
use ailoy::{agent::Agent, model::{APISpecification, InferenceConfig, LangModel}, value::{Message, Part, Role}};
use futures::StreamExt as _;

#[tokio::main]
async fn main() {
    let lm = LangModel::new_stream_api(APISpecification::OpenAI, "gpt-4o", "YOUR_API_TOKEN_KEY");
    let mut agent = Agent::new(lm, vec![]);
    let query = Message::new(Role::User).with_contents([Part::text("Please give me a short poem about AI")]);
    let mut strm = agent.run(vec![query], vec![], vec![], InferenceConfig::default());
    while let Some(result) = strm.next().await {
        if let Some(aggregated) = resp.aggregated {
            println!("{:?}", aggregated);
        }
    }
}
```

### Using local models

Here, we use the open-source model `Qwen3-0.6B`, running entirely on your own device ‚Äî no API keys or internet connection required.

<CodeTabs>

```python
import ailoy as ai
import asyncio

async def main():
    lm = await ai.LangModel.CreateLocal(model_name="Qwen/Qwen3-0.6B", progress_callback=print)
    agent = ai.Agent(lm)
    query = ai.Message(role=ai.Role.User, contents=[ai.Part.Text("Please give me a short poem about AI")])
    async for resp in agent.run([query]):
        if resp.aggregated:
            print(resp.aggregated.contents[0].text)

if __name__ == "__main__":
    asyncio.run(main())
```

```typescript
import { Agent, LangModel } from "ailoy";
import type { Message } from "ailoy";

async function main() {
    const lm = await ailoy.LangModel.newLocal(modelName="Qwen/Qwen3-0.6B", progressCallback=console.log);
    let mut agent = Agent::new(lm, vec![]);
    const query: Message = {role: "user", contents: [{ type: "text", text: "What is your name?" }]};
    for await (const resp of agent.run([query])) {
        if (resp.aggregated) {
            console.log(resp.aggregated.contents[0].text);
        }
    }
}

main().catch((err) => {
  console.error("Error:", err);
});
```

```rust
use ailoy::{agent::Agent, model::{InferenceConfig, LangModel}, value::{Message, Part, Role}};
use futures::StreamExt as _;

#[tokio::main]
async fn main() {
    let lm = LangModel::try_new("Qwen/Qwen3-0.6B").await.unwrap();
    let mut agent = Agent::new(lm, vec![]);
    let query = Message::new(Role::User).with_contents([Part::text("Please give me a short poem about AI")]);
    let mut strm = agent.infer(vec![query], vec![], vec![], InferenceConfig::default());
    while let Some(result) = strm.next().await {
        if let Some(aggregated) = resp.aggregated {
            println!("{:?}", aggregated);
        }
    }
}
```

:::info 
Don't be surprised if the output changes each time you run it.

An LLM's output includes a certain level of randomness, controlled by the `temperature` and `top_p`.
In ailoy, it can be adjusted with `InferenceConfig`.

If both `temperature` and `top_p` are set to 0, no randomness will occur.
